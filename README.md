# Icon-project-page

A simple Jekyll template for my icon project pages.

# Installation

Super quick.
* `git clone https://github.com/AdamWhitcroft/icon-project-page.git`
* `cd icon-project-page`
* `jekyll serve --watch`

# Usage

* Clone.
* Update as necessary.
* Copy contents of `_site` to a new folder on `adamwhitcroft.github.io`.
